Randomized Selection

回顾快速排序，在Partion之后，pivot左边的元素都小于pivot，pivot右边的元素都大于pivot，所以可以判断出pivot从小到大的次序[Math Processing Error]j（等于小于pivot的元素个数加[Math Processing Error]1）：

  • 如果[Math Processing Error]i<j，那么在小于pivot的[Math Processing Error]j−1个元素中找第[Math Processing Error]i小的元素。

  • 如果[Math Processing Error]i>j，那么在大于pivot的[Math Processing Error]n−j个元素中找第[Math Processing Error]i−j个元素。
